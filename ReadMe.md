# Project 06 - MongoDB Connection + DotEnv Practicing

## Setup

1. Copy `env.example` -> `.env` and replace with your MONGODB URI
2. Install dependencies : `npm install`
3. Run: `node index.js`

## Project Description

This project is a practice program on how to connect with MongoDB using its Node.js library Mongoose
The project uses Coding Language Node.js
NPM as a package manager
DotEnv is also used so that environment variables can be managed
